# Hospital Decision Support System

This machine learning application predicts expected hospital stay duration, treatment category, and medical bill range based on patient information.  
The system uses patient age, gender, department, and diagnosis as input features and applies classification models for prediction.  
The application is developed using Python, Scikit-learn, Joblib, and Streamlit for an interactive user interface.  
It is designed to support hospital planning and improve communication between healthcare providers and patients.
